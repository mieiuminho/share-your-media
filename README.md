## :rocket: Getting Started

### :hammer: Development

Starting the development server on port 5000.

```
mvn -Dserver.port=5000 spring-boot:run
```

Running tests.

```
mvn test
```

### :package: Deployment

Bundling the app into jar file.

```
mvn package
```
